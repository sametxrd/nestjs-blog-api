## 📖 Project Description

This project is a **NestJS TypeScript starter template** enhanced with:

- **HTTP Exception Filters** – for centralized error handling.
- **Validation Pipes** – for robust input validation.
- **Guards** – JWT authentication and role-based access control.
- **Interceptors** – response transformation, logging, and timing.
- **TypeORM** – SQL database integration with BaseEntity & Repository patterns.
- **JWT** – secure authentication for APIs.
- **Logger** – structured logging for requests and responses.
- **Custom BaseResponse format** – consistent API response structure.

This makes the application **secure, scalable, and maintainable**.
